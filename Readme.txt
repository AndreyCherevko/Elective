Выполненое задание: Система Факультатив. Существует перечень Курсов, за каждым из которых закреплен один Преподаватель. Студент записывается на один или несколько Курсов, данные о регистрации сохраняются. По окончании обучения Преподаватель выставляет Студенту оценку, которая сохраняется в Архиве.

Для запуска проекта необходимо:

0.Создать БД с помощью electiveDB.sql, что лежит в папке с проектом
1. Подключить к удаленному репозиторию и загрузить себе проект.
2.В IDEA: Run>Edit Configuration>+>Add TomcatServer
3.Deployment>+>указать dist\Elective.war
4.Run
